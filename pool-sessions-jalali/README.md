# Pool Sessions (Jalali) - افزونه مدیریت سانس‌های استخر

افزونه حرفه‌ای وردپرس برای مدیریت سانس‌های استخر با پشتیبانی کامل از تقویم شمسی (جلالی)، فیلتر جنسیت و سرویس، و قابلیت ایمپورت CSV/ICS.

## ویژگی‌های کلیدی

- **تقویم شمسی کامل**: پشتیبانی از تقویم جلالی با تبدیل خودکار تاریخ‌ها
- **فیلتر پیشرفته**: فیلتر بر اساس جنسیت (آقا/خانم/هردو) و سرویس‌ها
- **رابط کاربری ریسپانسیو**: طراحی مینیمال و موبایل‌فرست
- **ایمپورت داده**: پشتیبانی از فایل‌های CSV و ICS
- **مدیریت سرویس‌ها**: تعریف و مدیریت سرویس‌ها با رنگ‌های اختصاصی
- **تنظیمات پیشرفته**: شخصی‌سازی کامل ظاهر و رفتار تقویم
- **پشتیبانی RTL**: سازگار کامل با زبان فارسی

## پیش‌نیازها

- وردپرس 6.0 یا بالاتر
- PHP 8.0 یا بالاتر
- MySQL 5.7 یا بالاتر

## نصب

### نصب از فایل ZIP

1. فایل `pool-sessions-jalali.zip` را دانلود کنید
2. در پنل ادمین وردپرس، به بخش **افزونه‌ها > افزودن** بروید
3. روی **بارگذاری افزونه** کلیک کنید
4. فایل ZIP را انتخاب کرده و **اکنون نصب کن** را بزنید
5. افزونه را فعال کنید

### نصب دستی

1. محتویات پوشه را در مسیر `/wp-content/plugins/pool-sessions-jalali/` کپی کنید
2. از پنل ادمین، افزونه را فعال کنید

## استفاده

### شورت‌کد اصلی

```
[pool_calendar]
```

### پارامترهای شورت‌کد

- `gender`: جنسیت (male/female/all) - پیش‌فرض: all
- `service`: سرویس خاص یا * برای همه - پیش‌فرض: *
- `initial_year`: سال شمسی شروع - پیش‌فرض: سال جاری
- `initial_month`: ماه شمسی شروع - پیش‌فرض: ماه جاری

### نمونه‌های استفاده

```
[pool_calendar gender="male"]
[pool_calendar service="pool" gender="female"]
[pool_calendar initial_year="1404" initial_month="6"]
```

## تنظیمات

### پنل ادمین

پس از نصب، منوی **Pool Sessions** در پنل ادمین وردپرس ظاهر می‌شود که شامل:

#### ۱. تنظیمات عمومی
- منطقه زمانی نمایش
- روز شروع هفته (شنبه/یکشنبه)
- قالب نمایش زمان
- فعال/غیرفعال کردن tooltip و هدر تقویم
- پشتیبانی از جسچر موبایل

#### ۲. رنگ‌ها و ظاهر
- رنگ‌های اختصاصی برای جنسیت‌ها
- پشتیبانی از Dark Mode
- اندازه فونت و فاصله‌ها
- CSS سفارشی

#### ۳. مدیریت سرویس‌ها
- افزودن/ویرایش/حذف سرویس‌ها
- تعریف رنگ اختصاصی برای هر سرویس
- مدیریت اسلاگ و نام نمایشی

#### ۴. ایمپورت/اکسپورت
- ایمپورت CSV با نگاشت ستون‌ها
- ایمپورت ICS (فایل‌های تقویم)
- اکسپورت تنظیمات به JSON
- ایمپورت مجدد تنظیمات

## فرمت CSV

فایل CSV باید شامل ستون‌های زیر باشد:

```csv
service,gender,date,start,end,note,capacity
pool,male,1404-06-05,16:00,22:00,"سانس عصر",20
massage,female,1404-06-05,20:00,24:00,"ماساژ شبانه",10
```

### توضیحات ستون‌ها

- `service`: نام سرویس
- `gender`: جنسیت (male/female یا m/f)
- `date`: تاریخ شمسی (YYYY-MM-DD)
- `start`: زمان شروع (HH:MM)
- `end`: زمان پایان (HH:MM)
- `note`: یادداشت (اختیاری)
- `capacity`: ظرفیت (اختیاری)

## API

### REST API Endpoints

- `GET /wp-json/pool/v1/sessions` - دریافت سانس‌ها
- `GET /wp-json/pool/v1/services` - دریافت سرویس‌ها
- `POST /wp-json/pool/v1/import/csv` - ایمپورت CSV
- `POST /wp-json/pool/v1/import/ics` - ایمپورت ICS

### نمونه درخواست API

```javascript
fetch('/wp-json/pool/v1/sessions?year=1404&month=6&gender=male')
  .then(response => response.json())
  .then(data => console.log(data));
```
