## نحوه نصب

### ۱. نصب در وردپرس
1. فایل `pool-sessions-jalali.zip` را دانلود کنید
2. در پنل ادمین: **افزونه‌ها > افزودن > بارگذاری افزونه**
3. فایل ZIP را انتخاب کرده و نصب کنید
4. افزونه را فعال کنید

### ۲. استفاده
```
[pool_calendar]
[pool_calendar gender="male"]
[pool_calendar service="pool" gender="female"]
```

### ۳. تنظیمات
- منوی **Pool Sessions** در پنل ادمین
- تنظیمات عمومی، رنگ‌ها، سرویس‌ها
- ایمپورت/اکسپورت داده
